This week I spent a little more time on earthball adding the ability to generate a global atom & json feed for a site.

Links

  • desm is a nice Node.js wrapper module for getting the dirname from import.meta.url. This solves one of the more annoying aspects of moving to ecmascript modules: there's no longer a __dirname variable.
  • looom is a fun tool for creating animations on ipad and may be perfect for creating animated gifs that act as story illustrations or graphics for marketing websites.
    • Creating canvas or webgl exports of looom animations might be possible with this looom-tools project.
  • Hydra is an interesting tool for live coding visuals that can be used with p5.js.
